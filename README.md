# Portfolio-Chatbot-Local-RAG-Assistant-with-Llama-3.2-3B-Llama-3.1-8B
Developed a RAG-powered portfolio chatbot using Llama 3.2 (Ollama) + Llama 3.1 8B, FastEmbed, and NumPy that answers recruiter queries from a curated knowledge base while resisting prompt injection and preventing hallucinations through grounded retrieval.
